EcoSorter

Описание
EcoSorter - это приложение для сортировки мусора, предназначенное для помощи пользователям в правильной сортировке отходов. Приложение предоставляет удобный интерфейс для входа, регистрации, отслеживания статистики сбора мусора и даже взаимодействия с картой мест сбора отходов.

Возможности
Легкий доступ:

Простой интерфейс входа и регистрации для пользователей.
Отслеживание статистики:

Возможность просмотра статистики сбора мусора, накопленных EcoCoins и другой важной информации.
Интерактивная карта:

Взаимодействие с интерактивной картой для поиска и маркировки мест сбора мусора.
Безопасность:

Защита личной информации пользователей.
Использование
Вход и Регистрация:

Зарегистрируйтесь или войдите, используя свой аккаунт.
Интерактивная Карта:

Используйте карту для поиска ближайших точек сбора мусора.
Статистика и Профиль:

Просматривайте свою статистику и обновляйте свой профиль.
Сбор и Утилизация:

Собирайте мусор и утилизируйте его, зарабатывая EcoCoins.
Требования к Установке
Убедитесь, что на вашем устройстве установлен Python.
Установите необходимые зависимости с помощью pip install -r requirements.txt.
Запуск Приложения
bash
Copy code
python your_script.py
Замените your_script.py на имя вашего Python-скрипта.
